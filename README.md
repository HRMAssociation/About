Hirakahna Royal Magic Association (HRMA)，是一个主要成员位于极东地区的魔法研究、应用和推广的组织。  
我们希望魔法的光辉照亮整个地球，任何人都能便利地随时使用魔法，创造旧时代无法想象奇迹。  
我们希望每个人都可以实现愿望而不依赖或者单纯寄希望于某个特定的人，而更在于自身以及自己手中的魔导器。
